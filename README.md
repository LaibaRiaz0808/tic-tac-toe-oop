# 🎮 Tic Tac Toe Game (OOP)

## About

This project is a Tic Tac Toe game developed in Python using Object-Oriented Programming (OOP). It allows a human player to compete against a smart AI opponent built using simple decision-making logic.

## Features

- Human vs AI
- Smart AI
- Winner Detection
- Draw Detection
- GUI using Tkinter

## OOP Concepts

- Abstract Class
- Inheritance
- Polymorphism
- Encapsulation
- Method Overriding

## Technologies

- Python
- Tkinter
